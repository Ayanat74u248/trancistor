# 🎥 YouTube to Text - Транскрибер видео

Веб-приложение для получения текстовой стенограммы из видео YouTube по ссылке.

## ✨ Возможности

- 🎬 Загрузка видео с YouTube по ссылке
- 🎵 Автоматическое извлечение аудиодорожки
- 📝 Преобразование речи в текст (транскрипция)
- 🌍 Поддержка многих языков
- 📋 Копирование текста в буфер обмена
- 🎨 Красивый и удобный интерфейс

## 🚀 Установка

### 1. Установите Python 3.8+

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

**Важно:** Первый запуск Whisper загрузит модель (~1.4 GB), это может занять время.

### 3. Установите FFmpeg (для обработки аудио)

**Windows:**
```bash
# Используя Chocolatey
choco install ffmpeg

# Или используя winget
winget install FFmpeg.FFmpeg
```

**macOS:**
```bash
brew install ffmpeg
```

**Linux (Ubuntu/Debian):**
```bash
sudo apt-get install ffmpeg
```

## 💻 Запуск

### Backend (терминал 1):
```bash
cd backend
python -m uvicorn main:app --reload
```

Сервер запустится на `http://127.0.0.1:8000`

### Frontend (терминал 2):

1. Откройте файл `frontend/index.html` в браузере
2. Или используйте простой HTTP сервер:

```bash
cd frontend
python -m http.server 8001
```

Затем откройте `http://127.0.0.1:8001`

## 📖 Использование

1. Откройте приложение в браузере
2. Вставьте ссылку на YouTube видео
3. Нажмите "Получить текст"
4. Подождите обработки (время зависит от длины видео)
5. Скопируйте полученный текст кнопкой "Копировать текст"

## ⚙️ API Endpoint

**POST** `/transcribe`

Request:
```json
{
  "url": "https://www.youtube.com/watch?v=..."
}
```

Response:
```json
{
  "status": "success",
  "text": "Текст стенограммы...",
  "language": "en"
}
```

## ⚠️ Важные замечания

- **Время обработки:** Зависит от длины видео и мощности компьютера. Для 10-минутного видео это может занять 5-15 минут.
- **Качество транскрипции:** Зависит от качества аудио в видео
- **Языки:** Whisper поддерживает 99+ языков
- **Лимиты:** Нет встроенных ограничений, но помните о лимитах YouTube и его условиях использования

## 🐛 Решение проблем

### FFmpeg не найден
Убедитесь, что FFmpeg установлен и добавлен в PATH.

### CORS ошибки
Убедитесь, что backend запущен на `http://127.0.0.1:8000`

### Модель не загружается
Проверьте интернет-соединение и достаточно ли свободного места на диске (~2GB)
